# 山东华商汇理科技有限公司 网站规格文档

## 1. 项目概述

- **项目名称**: 山东华商汇理科技有限公司 官方网站
- **项目类型**: 单页展示型网站
- **核心功能**: 展示公司形象、业务范围、成功案例及业务规模数据
- **目标用户**: 潜在客户、合作伙伴、投资者

## 2. UI/UX 规格

### 2.1 布局结构

**页面 sections（从上到下）:**
1. **导航栏** - 固定顶部，包含logo和导航菜单
2. **Hero 区域** - 全屏首屏，展示公司口号和背景
3. **关于我们** - 公司介绍区域
4. **主营业务** - 汽车租赁服务介绍
5. **成功案例** - 组团送案例展示
6. **业务规模** - 数据统计展示
7. **联系我们** - 底部联系信息
8. **页脚** - 版权信息

**响应式断点:**
- 桌面端: >= 1024px
- 平板端: 768px - 1023px
- 移动端: < 768px

### 2.2 视觉设计

**配色方案:**
- 主色: #1a5fb4 (深蓝色 - 专业稳重)
- 辅助色: #26a269 (绿色 - 发展活力)
- 强调色: #e5a50a (金色 - 高端品质)
- 背景色: #fafbfc (浅灰白)
- 深色背景: #1e2a38 (深灰蓝)
- 文字色: #333333
- 次要文字: #666666

**字体:**
- 标题字体: "Noto Sans SC", "Microsoft YaHei", sans-serif
- 正文字体: "Noto Sans SC", "Microsoft YaHei", sans-serif
- 英文点缀: "Poppins", sans-serif

**字号:**
- 大标题: 48px
- 章节标题: 36px
- 小标题: 24px
- 正文: 16px
- 小字: 14px

**间距系统:**
- 章节间距: 80px
- 元素间距: 24px
- 卡片内边距: 24px

**视觉效果:**
- 卡片阴影: 0 4px 20px rgba(0,0,0,0.08)
- 悬停阴影: 0 8px 30px rgba(0,0,0,0.12)
- 过渡动画: all 0.3s ease

### 2.3 组件列表

**导航栏:**
- Logo + 公司名称
- 导航链接：首页、关于我们、业务介绍、成功案例、联系我们
- 移动端：汉堡菜单

**Hero 区域:**
- 全屏背景（渐变色 + 抽象几何图形）
- 主标题 + 副标题
-  CTA 按钮

**关于我们卡片:**
- 图文结合布局
- 公司简介文字
- 展示图片

**业务介绍卡片:**
- 图标 + 标题 + 描述
- 3列网格布局

**成功案例卡片:**
- 图片 + 标题 + 描述 + 客户名称
- 悬停效果：放大图片

**数据统计:**
- 大数字 + 标签
- 4列网格，数字动画计数

**联系信息:**
- 电话、地址、邮箱
- 二维码占位

## 3. 功能规格

### 3.1 核心功能

1. **平滑滚动导航** - 点击导航链接平滑滚动到对应区域
2. **数字动画** - 业务规模数据进入视野时触发数字递增动画
3. **卡片悬停效果** - 鼠标悬停时微微上浮并加深阴影
4. **响应式布局** - 适配不同屏幕尺寸
5. **移动端菜单** - 汉堡菜单展开/收起
